# 🛍️ Retail Store Sales Analysis

An interactive **Retail Store Sales Dashboard** built entirely in **Microsoft Excel** to analyze sales performance, customer behavior, regional performance, and order trends. The dashboard transforms raw retail data into meaningful business insights using Excel's analytical and visualization capabilities.

---

## 📌 Project Overview

Businesses generate thousands of sales transactions every month, making it difficult to identify trends and make informed decisions.

This project demonstrates how raw retail sales data can be transformed into an interactive dashboard that helps stakeholders monitor KPIs, identify growth opportunities, and understand customer purchasing patterns—all within Microsoft Excel.

---

## 🎯 Business Objective

The primary objective of this project is to answer key business questions, including:

* Which sales channels generate the highest revenue?
* Which states contribute the most to total sales?
* Which customer age group purchases the most?
* How does customer purchasing differ by gender?
* How do monthly sales compare with order volume?
* What is the distribution of order status (Delivered, Cancelled, Returned, etc.)?
* Which customer segments should be targeted to maximize sales?
